first-class failure UX

Every desktop incident is rendered in the operator surface, not hidden in console logs. Each state closes the release gate and keeps host_mutation=false.

SKIP · qemu unavailablepreflight blocks VM launch before any host mutation.
REFUSE · verifier rejectverifier evidence failed; attach is not attempted.
INCIDENT · lost streammalformed or empty daemon stream is unsafe to assume.
INCIDENT · timeouthung stream terminates child and requires cleanup proof.
INCIDENT · rollback failurerollback did not prove restored guest state.
INCIDENT · cleanup residueprocess or scratch residue remains visible.
REFUSE · duplicate idreplayed action ids are refused by the controller.
